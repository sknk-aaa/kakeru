# カケル（Kakeru）プロダクト・宣伝ブリーフ

> カケルの目的・機能・ユーザー体験・宣伝時の訴求軸をまとめたドキュメントです。  
> 広告、LP、SNS投稿、キャッチコピー、ユーザー向け説明をAIに相談するときの共有資料として使えます。

---

## 1. アプリ概要

**カケル（Kakeru）**は、**走らなければ課金されるランニング習慣化アプリ**です。

ユーザーはあらかじめランニング目標と罰金額を設定します。  
当日23:59までに目標を達成できなかった場合、登録したクレジットカードに自動で罰金が課金されます。

「走れば健康になる」だけでは続かなかった人に対して、  
「走らなければお金を失う」という仕組みを作ることで、行動を後押しします。

- サービスURL: https://www.kakeruapp.com
- 対応環境: スマートフォンブラウザ（iOS / Android）
- PWA対応: ホーム画面に追加してアプリのように利用可能

---

## 2. ターゲットユーザー

カケルが特に刺さりやすいユーザーは、次のような人です。

- ランニングを始めたいが、いつも三日坊主になってしまう人
- 健康のために走りたいが、「今日はいいか」と先延ばししてしまう人
- 意志力やモチベーションだけでは継続できないと感じている人
- 普通のランニングアプリや記録アプリでは行動につながらなかった人
- 自分で決めた約束を守るために、あえてプレッシャーがほしい人
- 走っているが、モチベーションが安定しない人

---

## 3. 解決する課題

多くの人は、ランニングのメリットを理解していても続きません。

- 健康になりたい
- 体力をつけたい
- 痩せたい
- 生活リズムを整えたい
- 運動習慣を作りたい

こうした目的があっても、実際には「今日は疲れた」「明日からでいい」「雨が降りそう」などの理由で行動が止まりがちです。

カケルは、行動できない理由を精神論で責めるのではなく、**走る理由が発生する仕組み**を作ります。

---

## 4. コアな価値・差別化

### 一番伝えたい価値

カケルの本質は「罰金アプリ」ではなく、**意志に頼らず、仕組みで走り続けるアプリ**です。

人は「何かを得ること」よりも「何かを失うこと」に強く反応します。  
カケルはこの損失回避の心理を使い、サボることに明確なコストを作ります。

### 他のランニングアプリとの違い

一般的なランニングアプリは、走った後の記録や分析が中心です。  
カケルは、走る前の「面倒くさい」「今日はやめたい」という気持ちに対して、行動する理由を作る点が特徴です。

### 宣伝で強調したい考え方

- モチベーションではなく、仕組みで続ける
- 走れば無料、サボれば罰金
- 自分に甘い人のためのランニング習慣化
- 逃げ道を先に塞ぐことで、未来の自分を助ける
- 本気で変わりたい人のためのコミットメントツール

## 5. 基本的な使い方

1. アカウントを登録する
2. クレジットカードを登録する
3. ランニング目標を設定する
4. スマホを持って走る
5. GPSで距離や時間を計測する
6. 目標を達成すると記録される
7. 23:59までに未達成の場合、設定した罰金が自動で課金される

目標を設定しない限り罰金は発生しません。  
走り続けて目標を達成していれば、罰金は発生しません。

---

## 6. 主な機能

### ランニング計測

- スマートフォンのGPSで走行距離を計測
- 距離、時間、ペース、カロリーを表示
- 走ったルートを地図上に表示

### 目標設定

| 目標の種類 | 内容 |
|-----------|------|
| 繰り返し目標 | 毎週特定の曜日に走る目標。例: 毎週月・水・金に5km |
| 単発目標 | 特定の日だけ走る目標。例: 今週土曜日に10km |
| チャレンジ目標 | 一定期間内の累積距離・時間を目標にするPRO限定の目標 |

- 距離または時間で目標を設定可能
- 罰金額は目標ごとに設定可能

### 課金・罰金

- 当日23:59に未達成の目標を自動判定
- 未達成の場合、設定した罰金額を自動課金
- 過去の罰金履歴を確認可能
- カード情報はStripeが管理し、カケルのサーバーには保存しない

### スキップ

- 月1回、罰金なしで目標をスキップ可能
- 居住地域が雨の場合、通常のスキップ回数を消費せずに雨天スキップ可能

### 記録・統計

- 今週の目標と達成状況を確認
- 月間走行距離、達成率、累計罰金額を確認
- 過去のランニング履歴を確認

### 通知・リマインダー

- 朝と夜のリマインダーを設定可能
- 未達成の目標がある日に、走るきっかけを作る

### PWA対応

- 専用アプリのインストールは不要
- スマートフォンのホーム画面に追加して、アプリのように起動できる

## 7. 料金プラン

| | FREE | PRO |
|--|------|-----|
| GPSランニング計測 | 利用可能 | 利用可能 |
| 繰り返し・単発目標 | 利用可能 | 利用可能 |
| 月1回スキップ | 利用可能 | 利用可能 |
| 雨天スキップ | 利用可能 | 利用可能 |
| 記録・統計 | 利用可能 | 利用可能 |
| チャレンジ目標 | - | 利用可能 |
| エスカレーション | - | 利用可能 |
| クーリング期間 | - | 利用可能 |
| 目標ロック | - | 利用可能 |
| 月額 | 無料 | 480円/月 |
| 年額 | 無料 | 4,800円/年 |

PROは、より強い自己規制をしたい人向けのプランです。

### PRO限定機能

- **チャレンジ目標**: 一定期間内の累積距離・時間を目標にする
- **エスカレーション**: 連続で失敗すると罰金が増える
- **クーリング期間**: 一定期間、目標の変更や停止をできなくする
- **目標ロック**: 目標を変更・停止できない状態にする

PROはいつでも解約可能です。  
解約後も契約期間終了まではPRO機能を利用できます。

## 8. 安心材料

宣伝では、インパクトのある特徴とあわせて安心材料も伝える必要があります。

- 目標を設定しなければ罰金は発生しない
- 走って目標を達成すれば罰金は発生しない
- 月1回の通常スキップがある
- 雨天時は通常スキップ回数を消費せず休める
- カード情報はStripeが管理する
- まずは軽い目標から始められる
- 距離でも時間でも目標を作れる

### 自分に甘い人向け

- 自分に甘い人のためのランニングアプリ
- 「また明日」を終わらせる
- 意志が弱くても、仕組みで走れる

### 損失回避

- 走れば無料、サボれば罰金
- 失いたくないから走れる
- 健康になる喜びより、お金を失う痛みで動く

### 本気感・コミットメント

- 逃げ道を先に塞ぐ
- 未来の自分がサボれないようにする
- 本気で習慣を変えたい人へ

### ランニング初心者向け

- まずは軽い目標から始められる
- スマホだけで始められる
- 普通の記録アプリでは続かなかった人でも試しやすい

## 10. 宣伝時に避けたい表現

避けたい表現:

- 「必ず痩せる」
- 「絶対に習慣化できる」
- 「誰でも必ず続く」
- 「課金で追い込め」
- 「お金を失う恐怖で走れ」
- 「サボる人はダメ」
- 「罰金を払わせるアプリ」

注意点:

- 効果を断言しすぎない
- ユーザーを責める言い方にしない
- 課金への恐怖だけを煽らない
- 罰金の説明には、スキップやStripe管理などの安心材料をセットで伝える
- 健康・ダイエット効果を過度に保証しない

## 11. よくある質問

**Q. 本当にカードに引き落とされますか？**  
A. はい。当日23:59までに目標を達成しなかった場合、設定した罰金額が自動で引き落とされます。

**Q. 試しに使いたいだけのとき、課金されますか？**  
A. 目標を設定しない限り罰金は発生しません。走って目標を達成している限り、罰金も発生しません。

**Q. カード情報は安全ですか？**  
A. カード情報はカケルのサーバーには保存されません。決済サービスのStripeが管理します。

**Q. 体調不良や雨の日はどうなりますか？**  
A. 月1回のスキップ機能があります。また、居住地域の天気が雨の場合は、通常のスキップ回数を消費せずに休めます。

**Q. PROは解約できますか？**  
A. PROはいつでも解約できます。解約後も契約期間終了まではPRO機能を利用できます。
