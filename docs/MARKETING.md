# Marketing

カケルのLP、ストア/アプリ訴求、SNS投稿、動画台本、価格/PRO導線AB案を蓄積する場所。製品・宣伝ブリーフの正は `docs/KAKERU_PRODUCT.md`、現在の技術状態は各実装docを参照する。

## 目的

- 「走らなければ課金される」という強い特徴で注意を取り、信頼できる説明で登録まで進める。
- 実装相談でも、初見の伝わり方、登録CV、カード登録への不安、PRO購入、継続率への影響を同時に見る。

## 基本方針

- 本質は「罰金アプリ」ではなく「意志に頼らず、仕組みで走り続けるコミットメントツール」。
- 訴求は「走れば無料、サボれば罰金」「また明日を終わらせる」「自分に甘い人のためのランニング習慣化」。
- インパクト訴求には必ず安心材料を添える。目標を設定しなければ罰金は発生しない、達成すれば課金されない、月1回スキップと雨天スキップがある、カード情報はStripe管理。
- PRO訴求は、より強い自己規制をしたい人向けに、チャレンジ目標、エスカレーション、クーリング期間、目標ロックを整理して伝える。

## 避けること

- 「必ず痩せる」「絶対に習慣化できる」「誰でも続く」など成果を保証する表現。
- 課金で脅すだけの表現。信頼・透明性・コントロール感を失う訴求。
- 医療・健康効果を断定する表現。
